# 🤖 Gold Bot — Telegram бот для продажи голды

## Установка

### 1. Установи Python 3.11+
Скачай с https://python.org

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Настрой .env файл
Скопируй `.env.example` → `.env` и заполни:
```
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=твой_telegram_id
```

**Как получить токен:** напиши @BotFather в Telegram → /newbot

**Как узнать свой ID:** напиши @userinfobot

### 4. Запусти бота
```bash
python bot.py
```

---

## Команды администратора

| Команда | Описание |
|---------|----------|
| `/orders` | Список последних 10 заказов |
| `/done 5` | Пометить заказ #5 выполненным (клиент получит уведомление) |

---

## Как работает бот для клиента

1. Клиент пишет `/start`
2. Нажимает **💰 Купить голду**
3. Вводит нужную сумму (например `2000`)
4. Бот считает цену скина: `2000 × 1.20 = 2400 G`
5. Клиент выставляет скин за `2400 G`, делает скрин с аватаркой
6. Присылает скрин боту
7. Тебе в личку приходит уведомление со скрином и данными
8. Ты находишь скин по аватарке → покупаешь → отправляешь `/done <id>`
9. Клиент получает: **"Голда зачислена! 🎉"**
